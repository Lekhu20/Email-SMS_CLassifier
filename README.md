
# 📧 Email/SMS Spam Classifier

A lightweight Machine Learning web application built using Python, Streamlit, and Scikit-Learn that predicts whether an incoming message/email is **Spam** or **Ham (Safe)**.

## 🚀 Features
- **Live UI:** Built using Streamlit for an interactive user experience.
- **ML Model:** Uses Multinomial Naive Bayes classifier for text classification.
- **Vectorization:** Converts text into numerical data using CountVectorizer.
- **Fast & Lightweight:** Processes and predicts in real-time.

## 🛠️ Tech Stack
- Python
- Streamlit (Frontend UI)
- Pandas (Data Manipulation)
- Scikit-Learn (Machine Learning)
